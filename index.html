Mes Projets

Palendar

Développé en TypeScript et Tailwind CSS avec React pour la partie Front-End et Python FastAPI pour le Back-End. Palendar est un outil de calendrier collaboratif permettant à des amis et des proches d'organiser des événements en renseignant leurs disponibilités. Pour réaliser cette application nous avons utilisé la méthode Kanban afin d'organiser notre travail.

Messagerie directe

Développement d'une application web de messagerie instantanée utilisant une base de données PHP et des requêtes AJAX avec jQuery.

Kitchen-wizard

Application mobile développée en Java en utilisant l'IDE Android Studio. Cette application est reliée à une API de recettes en tout genre et vous permet de choisir la recette que vous souhaitez concocter. Si vous n'avez pas d'idée un bouton vous propose une recette aléatoire. Lorsque vous trouvez une recette vous pouvez l'ajouter à vos favoris pour la retrouver à tout moment.

Nuit de l'info

La Nuit de l'Info est un événement organisé à l'échelle nationale où les étudiants en informatique doivent relever un défi, réaliser une application en une nuit. J'ai eu l'occasion d'y participer à 3 reprises, en 2024 nous devions concevoir une application pour la fondation Race for Water.
